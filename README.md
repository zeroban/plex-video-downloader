# Plex Video Downloader

Python desktop application built with CustomTkinter for downloading and organizing videos.

## Screenshot

![Plex Video Downloader](screenshot.png)

## Features

- Download videos directly from URLs
- TV Show / Anime organization
- Movie support
- Multi-threaded downloads
- Configurable worker count
- Adjustable chunk size
- Download queue tracking
- Stop active downloads
- Plex-friendly file organization

## Requirements

- Python 3.12+
- pip

## Install

```bash
pip install -r requirements.txt
